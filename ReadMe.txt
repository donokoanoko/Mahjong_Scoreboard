・内容
    麻雀の点数の合計を計算して、OBSに表示してくれるソフト。
    おまけでOBS上に名前を表示する機能も有り。


・使い方(OBS)
    ソース→追加→テキスト(GDI+)を追加、「ファイルからの読み取り」にチェックを入れ"mahjong_score.txt"を指定。
    "設定"タブからプレイヤーの名前を設定と表示形式を設定しておけば終了ボタンを押すたびに更新されます。

    もしOBS上にネームプレートを表示したい場合は、ソース→追加→ブラウザを追加し、ローカルファイルとして"下家.html","上家.html","対面.html"の3つを追加し、色のついた部分のみが残るように表示範囲を変更し配置してください。


・インストール
    ZIPファイルを解凍するだけで利用できます。


・アンインストール
    フォルダごと消去するだけで可能です(レジストリなど利用していません)。
    設定ファイルが"C:\Users\｛ユーザ名｝\AppData\Local\Mahjong_Score"に保存されているので、気になる人はそちらも消去してください。



・各種説明
    [メイン]
        【開始(終了)】                ：【開始】を押すと設定しているならネームプレートが表示されます。【終了】を押すと、得点の計算をしてOBSの表示を更新してくれます。
        【リセット】                  ：全員の得点を0点にします。
        【得点】                      ：終了時の得点をここに入力します。【終了】ボタンを押したタイミングで反映されます。
        【ボーナス点】    　　　      ：加点したい時に使用します。すぐ右の【ボーナス】ボタンで個別に加算します。特殊ルール(○○するだけで+1000点)みたいな時に使います。
        【罰符】                      ：減点したい時に使用します。すぐ右の【罰符】ボタンで個別に加算します。特殊ルール(○○したら-1000点)みたいな時に使います。
    [設定]
        【プレイ人数】                ：4人麻雀と3人麻雀の切り替え用。
        【一位の人に王冠をつける】    ：1位の人のOBS上のネームプレートに絵文字の王冠(👑)をつけます。
        【サーバーアドレス】          ：サーバーのポートが被った時のみ変更してください。                                         
        【OBS用テキストファイル】     ：OBS上に表示するテキストの保存場所を指定します。空欄で"Mahjong_Score.exe"と同じ場所に作られます。
        【スコアボードの表示スタイル】：OBS上に表示するテキストの表示形式を指定します。タグを使った場合は自動で対応した物に置き換えられます。
        【プレイヤー名】              ：プレイヤーの名前を設定します。
        【プレイヤーカラー】          ：OBS上のネームプレートの背景色をここで設定します。
